# Скриншоты

Положите сюда PNG-скриншоты приложения, затем раскомментируйте ссылки на них
в корневом `README.md` (раздел «Скриншоты»).

Рекомендуемые файлы:

- `screenshot-main.png` — главный экран: сайдбар с настройками + выбор папки/промпта.
- `screenshot-processing.png` — процесс обработки: прогресс-бар, лог, превью.
- `screenshot-dataset.png` — вкладка работы с датасетом: статистика тегов / галерея.

Совет: делайте скрины на нейтральных изображениях, они будут видны публично.
